Detailed Description

Abstract base class for Manager components.

Managers coordinate business logic and multiple drivers/services.

Derived classes must implement:

  • GetModuleName(): Return human-readable name (e.g., "WaterMonitor")
  • OnInit(): Initialize manager resources
  • OnUpdate(): Periodic update logic

The parent Module handles Init() and Update() calls and logging. Manager::Update() additionally detects battery mode changes via PowerController.

Note
Managers typically require periodic updates. Override OnUpdate().
