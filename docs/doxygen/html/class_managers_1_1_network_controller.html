void Managers::NetworkController::ChangeState ( const State newState,
const int delayMs = -1 )
private

Updates the internal state of the NetworkController.

Parameters
newStateThe new state to transition to.
delayMsOptional delay in milliseconds before the state transition takes effect. Note: This method also starts the internal delay timer if a delay is specified. The OnUpdate() method will check this timer and perform the state transition when the delay has elapsed. This allows for non-blocking delays between state transitions.
