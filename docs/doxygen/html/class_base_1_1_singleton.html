Detailed Description

template<typename T>
class Base::Singleton< T >

Template base class providing Singleton pattern via CRTP.

Template Parameters
TThe derived class (CRTP - must be passed as template parameter)
Note
The derived class automatically inherits GetInstance() without needing to define it. Just inherit from Singleton<YourClass>.
Thread-safe in C++11 and later due to static local variable initialization guarantees.
