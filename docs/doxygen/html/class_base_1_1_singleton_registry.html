Detailed Description

Global registry for module instances.

This registry allows any module to be retrieved by type anywhere in the system. Each derived module class (Manager, Driver, Service) registers itself upon initialization, making it accessible via GetInstance<T>().

Example:

// In MyManager initialization
SingletonRegistry::Register<MyManager>(this);
// Later, anywhere in the code
auto* manager = SingletonRegistry::Get<MyManager>();
