Detailed Description

Example Manager implementation showing the GetInstance pattern.

This example demonstrates:

  1. How to register the module instance during initialization
  2. How to retrieve other modules using GetInstance<T>()
  3. How derived classes access shared modules without type casting

Usage pattern:

// Create and initialize
MyExampleManager manager;
manager.Init(); // Automatically registers itself
// Retrieve from anywhere else in code
auto* myManager = Base::Manager::GetInstance<MyExampleManager>();
if (myManager) {
myManager->DoWork();
}
// Retrieve other module types (if they're registered)
auto* someDriver = Base::Manager::GetInstance<SomeDriverType>();
